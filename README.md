# ALS Online Enrollment and E-Learning System

This project is a web-based system designed for Alternative Learning System (ALS). It provides an online enrollment process and an e-learning platform where students can access lessons, modules, and learning materials anytime.

The system helps streamline enrollment by allowing students to register online without going to the school physically. It also helps instructors manage students and upload learning resources efficiently.

Built using PHP, MySQL, HTML, CSS, and JavaScript, this system aims to improve accessibility and digital learning experience for ALS students.

Features:
- Online Enrollment System
- Student Management
- E-Learning Module Access
- Admin Dashboard
- Database Integration

This project is developed as a portfolio requirement for Information Technology studies.
